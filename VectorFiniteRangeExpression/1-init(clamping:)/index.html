Generic Initializer

init(clamping:)

Converts a finite integer vector range with bounds with elements of type U to a finite integer vector range with bounds with elements of type T, with each element of each bound clamped to the range of values representable by T.

Available when Storage is SIMD3<T> and T conforms to FixedWidthInteger.

Declaration
